Voice-to-Text Web Application

This project is a web-based application that allows users to record their voice through a dedicated button on the web page and convert the recorded audio into text in real time.

The converted text is instantly displayed on the same page, providing a smooth and interactive user experience. In addition, the generated text is automatically stored in the connected database for persistence and future use.

Key Features

Voice recording through a web interface

Real-time speech-to-text conversion

Instant display of converted text on the page

Automatic storage of text data in the database

Seamless integration between frontend and backend
